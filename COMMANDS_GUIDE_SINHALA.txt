• මේකේ commands ඔක්කොම SLASH COMMANDS. ඒ කියන්නේ Discord එකේ chat box
  එකේ "/" කියලා type කරාම commands list එක එනවා. පරණ වගේ "!ban" වගේ
  prefix commands නෑ — හැමදේම "/" වලින් පටන් ගන්නවා.

• සමහර commands වලට "permission" ඕන (උදා: Ban Members, Administrator).
  ඒ permission නැත්නම් command එක ඔයාට Discord එකේ පේන්නෙවත් නෑ, නැත්නම්
  "You don't have permission" කියලා පණිවිඩයක් එනවා.

• commands දෙ ආකාරයක් තියෙනවා:
    1) සාමාන්‍ය commands  →  /ping, /ban වගේ තනි ඒවා.
    2) Group commands     →  /warn add, /security antinuke වගේ. මුල් වචනය
       group එකක් (උදා: warn), දෙවැනි වචනය ඒකේ උප-command එකක් (add).

• AUTOMATIC ආරක්ෂාව: Anti-Nuke සහ Anti-Raid කියන දෙක commands නෙවෙයි.
  ඒවා background එකේ ඉබේම වැඩ කරනවා (කවුරුහරි server එක නස්ති කරන්න
  හදනකොට ඉබේම අල්ලනවා). ඒවා on/off කරන්නේ /security commands වලින්.

  ▸ /warn add
      මොකද කරන්නේ : සාමාජිකයෙක්ට warning එකක් දෙනවා. එයාට DM එකකුත්
                     යනවා (DM open නම්).
      ඕන දේවල්    : member (කවුද), reason (හේතුව — අනිවාර්යයි)
      Permission  : Moderate Members
      උදාහරණය     : /warn add member:@Kasun reason:නරක වචන

  ▸ /warn list
      මොකද කරන්නේ : කෙනෙක්ට දීලා තියෙන warnings ඔක්කොම පෙන්නනවා
                     (ID එක්කම).
      ඕන දේවල්    : member (කවුද)
      Permission  : Moderate Members
      උදාහරණය     : /warn list member:@Kasun

  ▸ /warn remove
      මොකද කරන්නේ : තනි warning එකක් ID එකෙන් අයින් කරනවා.

      උදාහරණය     : /warn remove warning_id:5

  ▸ /warn clear
      මොකද කරන්නේ : කෙනෙකුගේ warnings ඔක්කොම එකපාරට මකනවා.

        --> moderation actions log වෙන තැන දාන්න.

  2) /security setlog channel:#security-logs
        --> ආරක්ෂක (private) logs යවන තැන දාන්න.

  3) /security setalerts channel:#alerts
        --> public threat announcements යවන තැන දාන්න.

  4) /security antinuke enabled:True
        --> Anti-Nuke on කරන්න.

  5) /security antiraid enabled:True
        --> Anti-Raid on කරන්න.

  6) /security minage hours:168
        --> අලුත් accounts filter කරන්න (දවස් 7ක්).

  7) /verification setup channel:#verify verified_role:@Member
        --> verification gate එක on කරන්න.

  8) /backup now
        --> මුල්ම backup එක ගන්න.

  9) /trust add member:@YourCoOwner   (ඕන නම් විතරක්)
        --> විශ්වාසවන්ත admins ලා trust list එකට දාන්න.

  10) /securitystatus
        --> ඔක්කොම හරියට on වෙලාද කියලා බලන්න.
